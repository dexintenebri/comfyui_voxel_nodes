A collection of powerful 3D/voxel custom nodes for ComfyUI, including:

- depth_to_vox.py: Convert depth maps to voxel volumes
- multi_view_voxel_fusion.py: Fuse multi-angle depth/rgb maps into a single voxel model
- wfc_terrain_generator.py: WFC procedural 3D terrain generation from image using Wave Function Collapse
- midvoxio: Voxel export and visualization library
- voxel_nodes.py: Load image, depth maps to voxel, attempt to make 3d voxel model

2. Install dependencies:

   pip install -r requirements.txt
